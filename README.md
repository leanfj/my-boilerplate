### Installation

First of all, install the dependencies to run this boilerplate.

- [NodeJS](http://nodejs.org/)
- [GulpJS](http://gulpjs.com/)


```sh
# Clone this repository
$ git clone git@github.com:Qualy-org/qualy-front.git
$ cd qualy-front

# install dependencies
$ npm install
```
