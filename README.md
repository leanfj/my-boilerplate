### Installation

First of all, install the dependencies to run this boilerplate.

- [NodeJS](http://nodejs.org/)
- [GulpJS](http://gulpjs.com/)


```sh
# Clone this repository
$ git clone https://github.com/leanfj/my-boilerplate.git
$ cd my-boilerplate

# install dependencies
$ npm install
```
